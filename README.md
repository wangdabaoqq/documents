## 安装
```sh
# use npm
npm install vue-drag-tree-org

# use yarn
yarn add vue-drag-tree-org
```
## 使用
```js
# use
import VueDragTree from 'vue-drag-tree-org'

Vue.use(VueDragTree)

# or use
compoents: {
  VueDragTree
}
```
## CDN
```html
# css
<link href="https://unpkg.com/vue-drag-tree-org/lib/theme/index.css">
```
